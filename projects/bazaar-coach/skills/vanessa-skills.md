# Vanessa Skills — Complete Reference

**Last Updated**: 2026-02-21 (Patch 11.3)
**Total Skills**: ~188 (includes possible renames/reworks across patches)
**Sources**: bazaardb.gg, thebazaar.wiki.gg, mobalytics.gg, thebazaargame.net

## How To Use This File

Skills are organized by **category** so the coach can quickly find relevant skills for a board state. When evaluating a board, search by the dominant archetype tags (Weapon, Poison, Burn, Aquatic, etc.) to find applicable skills.

Values shown as `a / b / c / d` = Bronze / Silver / Gold / Diamond tier scaling.
Starting tier is noted where known. (UNIQUE) = Vanessa-only skill.

## Notes

- **Discrepancies**: Multiple sources show different tier values and starting rarities. The Bazaar has been actively patched through Season 4. BazaarDB tends to be most current; thebazaargame.net often has older data.
- **Duplicate names**: Some skills appear reworked across patches (e.g., Healthy Jolt / Overheal Haste may be the same skill renamed). The list may contain ~10 such duplicates.
- **Monster skills**: Some skills in this list are acquirable from PvE encounters (monster drops) and are shared across heroes, not Vanessa-exclusive.
