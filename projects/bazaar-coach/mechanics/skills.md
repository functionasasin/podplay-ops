# Skills System

## Overview

Skills are **passive abilities** that provide permanent bonuses throughout your run. Unlike items, skills do not occupy board slots -- they exist in a separate skill area and are always active. Skills are one of the three pillars of your build alongside items and enchantments.

## How Skills Work

- Skills provide **always-on passive effects** (stat bonuses, conditional triggers, scaling)
- They do not have cooldowns and do not trigger like items
- Effects include: crit boosts, damage bonuses, cooldown reduction, status effect application, healing, shielding, and economic benefits
- Some skills have **conditional triggers** (e.g., "The first time you fall below half health each fight")
- Skills **stack infinitely** for synergies -- having multiple complementary skills compounds their effects

## Skill Types

### Hero Skills
Unique to each hero. Acquired from hero-specific Skill Trainers at certain levels:
- **Vanessa**: Aquatic synergies, weapon buffs, ammo management
- **Dooley**: Tech synergies, single-item focus
- **Pygmalien**: Economy and value scaling
- **Mak**: Potion and reagent synergies
- **Stelle**: Weapon and vehicle synergies
- **Jules**: Various specialized effects

### Monster Skills
Acquired from PvE encounters. Available to all heroes regardless of hero choice. Some notable monster skills:
- **Augmented Weaponry** (Trashtown Mayor): Top-tier weapon scaling
- **Petrifying Gaze**: Board-wide enemy freezing
- Various damage, burn, poison, and defensive skills

### Shared Skills
Available across multiple heroes. Provide universal benefits like crit chance, cooldown reduction, or economic boosts.

## Skill Tiers

Skills follow the same tier progression as items:

| Tier      | Notes                                    |
|-----------|------------------------------------------|
| Bronze    | Basic effect                             |
| Silver    | Improved effect                          |
| Gold      | Strong effect                            |
| Diamond   | Very powerful effect                     |
| Legendary | Rarest and most powerful                 |

### Upgrading Skills

Same as items: acquire a **duplicate of the same skill at the same tier** to upgrade it one level. Each tier upgrade provides a **substantial boost** to the skill's effect.

## Skill Acquisition

### Skill Trainers (Level-Up Rewards)

| Trainer  | Tier    | First Available | Notes                          |
|----------|---------|-----------------|--------------------------------|
| Pip      | Bronze  | Level 2         | First skill trainer            |
| Argenta  | Silver  | Level 5         | Silver-tier skills             |
| Orlin    | Gold    | Level 8         | Gold-tier skills, major spike  |
| Adira    | Diamond | Level 14        | Diamond-tier skills, endgame   |

Hero-specific trainers also appear at certain levels (e.g., Cymon at Level 11, Mr. Tuskari at Level 17).

### Other Acquisition Methods

- **PvE encounters**: Monsters drop skills based on their type and tier
- **Events**: Random events sometimes offer skill choices
- **Start-of-run option**: The Skill Start gives you one monster skill immediately
- **Level-up rewards**: Some levels offer direct skill choices

## Skill Categories

Skills modify various mechanics:

| Category          | Examples                                              |
|-------------------|-------------------------------------------------------|
| Damage            | +X% damage, bonus damage per weapon, conditional damage |
| Burn/Poison       | Apply Burn/Poison on specific triggers                |
| Defensive         | +Shield, +Heal, +Regen on conditions                 |
| Speed             | Cooldown reduction, Haste effects                    |
| Economy           | Gold generation, sell value bonuses                   |
| Item Interaction  | Charge items, reload Ammo, upgrade items              |
| Positional        | Buff leftmost/rightmost item                          |
| Conditional       | "If you have exactly one Weapon...", "First time below half health..." |

## Skill Strategy

- **Don't skip Skill Trainers**: Skills are free power. Always evaluate what's offered.
- **Keep 2+ gold for skill rerolls**: At key level-ups, having gold to reroll the skill trainer can find build-defining skills.
- **Build around synergy**: A Poison skill is wasted if you have no Poison items. Match skills to your board theme.
- **Monster skills are universal**: A great monster skill from PvE can work with any hero. Don't pass on strong ones.
- **Level 8 (Orlin) is a power spike**: Gold-tier skills are significantly stronger than Silver. Prioritize reaching Level 8.
- **Level 14 (Adira) defines endgame**: Diamond skills are build-defining. Plan your XP to reach this level.
