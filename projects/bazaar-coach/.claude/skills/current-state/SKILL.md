# Current State Check

Before recommending ANY item purchase, upgrade, or build decision, you MUST first review the current board state. This prevents recommending items that don't synergize with what's actually on the board.

## Steps

1. **List the current board** (all items, their types, sizes, tiers, cooldowns):
   - How many are Weapons vs non-Weapons?
   - How many are Aquatic vs non-Aquatic?
   - How many are Friends?
   - How many have Burn/Poison/Slow/Freeze/Haste?
   - How many are passive vs cooldown-based?
   - What's the total slot usage?

2. **List the backpack** (items being held but not on board):
   - What's available to swap in?
   - What's available to sell?

3. **List active skills and enchantments**

4. **Note key stats**: Gold, income, prestige, day/hour, HP, Regen

5. **Identify what the board ACTUALLY needs**:
   - What's the primary engine? (e.g., Sharkclaws buffing weapons, Beach Ball hasting Aquatics)
   - What items are already self-sustaining and don't need more support?
   - What's the weakest slot on the board?
   - What type of item would have the highest marginal impact?

## When Evaluating a Shop Item

For EACH item being considered, answer:
- How many items on my current board does this interact with?
- Does this duplicate something already covered? (e.g., Haste on items already perma-hasted)
- What would I cut from the board to fit this?
- Is the gold better saved for a future shop?

## Common Mistakes to Avoid

- Recommending Haste items when key items are already perma-hasted
- Recommending non-Weapon synergy items when the board is 70%+ Weapons
- Recommending Aquatic enablers when the board is already mostly Aquatic
- Ignoring that passive items don't benefit from Haste
- Rating items by their KB tier (S/A/B) without checking if they fit the CURRENT board
- An A-tier item that synergizes with 1 board item is worse than a B-tier item that synergizes with 5
