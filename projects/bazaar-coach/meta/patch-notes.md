# Patch Notes

## Patch 11.0 (February 4, 2026) -- CURRENT PATCH (Season 11)

### Major System Changes
- **Burn rework**: Burn now decrements by 5% per tick (was flat 1). Little impact at low Burn, but reduces top-end output by ~25-30%. Significant nerf to Burn-focused strategies.
- **Shield vs Burn**: Shield reduction from Burn capped to current Shield value when Burn ticks.
- **Obsidian Enchantment rework**: No longer provides Lifesteal. Instead turns item into a Weapon or buffs damage if already a Weapon. Most items can now be Obsidian-enchanted.
- **Toxic/Fiery enchantments**: Effects doubled from 5% to 10% across nearly all Vanessa items.
- **Crash Site Expedition**: New content enabling unconventional builds with off-Hero items.
- **Repair mechanic**: New system restoring destroyed items (community considers it "over-tuned").

### Net Vanessa Impact
Weapon/Ammo builds strengthened. Burn builds took indirect hit from global Burn nerf. Meta shifted toward Weapon/Ammo aggro and away from pure Burn.
