## Instructions

   - 1-4 courts: 1TB SSD
   - 5-8 courts: 2TB SSD

2. Add network switch sizing logic that overrides the template quantity for the switch line item:
   - 1-8 courts: 24-port switch (qty 1)
   - 9-16 courts: 48-port switch (qty 1)
   - 17+ courts: 48-port switch (qty 2)
3. These overrides replace the default SSD and switch entries from the tier template with the correctly sized SKU and quantity.
4. Ensure the SKU changes at each boundary (different product, not just different quantity).
