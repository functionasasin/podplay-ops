# Stage 024 — Login Page (/login)

## Spec Files
- `ui-spec/routes.md`

## Instructions
1. Create `src/routes/login.tsx`.
2. Build an email/password login form using React Hook Form with a Zod schema (email: valid email, password: min 6 chars).
3. Add a "Sign in with magic link" option that calls `supabase.auth.signInWithOtp({ email })`.
4. Wire the primary submit to `supabase.auth.signInWithPassword({ email, password })`.
5. Display validation errors inline under each field.
6. On successful sign-in, redirect to `/projects`.
7. On error, show an error message (use toast or inline alert).
8. Register the route in the router config.
