# Stage 025 — Auth Callback (/auth/callback)

## Spec Files
- `ui-spec/routes.md`

## Instructions
1. Create `src/routes/auth/callback.tsx`.
2. On mount, extract the `code` query parameter from the URL.
3. Call `supabase.auth.exchangeCodeForSession(code)` to complete the PKCE exchange.
4. On success, redirect to `/projects`.
5. On failure (missing code, exchange error), redirect to `/login` with an error query param.
6. Show a loading spinner while the exchange is in progress.
7. Register the `/auth/callback` route in the router config.
