## Instructions
1. Create `src/lib/enum-labels.ts`.
2. For all 21 enum types defined in `ui-spec/enum-labels.md`, export three maps each:
   - `{enumName}Labels`: `Record<EnumValue, string>` — value to display label
   - `{enumName}BadgeClass`: `Record<EnumValue, string>` — value to badge CSS className
   - `{enumName}SortOrder`: `Record<EnumValue, number>` — value to sort order
3. Every label and className must match `enum-labels.md` exactly — do not invent labels.
4. Export a generic helper: `getEnumLabel(enumType: string, value: string): string` that looks up the correct map.
5. Export a generic helper: `getEnumBadgeClass(enumType: string, value: string): string`.
