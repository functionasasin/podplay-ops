# Stage 002 ‚Äî Tailwind 4 + shadcn/radix Setup

## Spec Files
- `deployment/infrastructure.md`

## Instructions
1. `cd apps/podplay`
2. Install Tailwind CSS v4 and the Vite plugin: `npm install tailwindcss@4 @tailwindcss/vite`
3. Add `@tailwindcss/vite` plugin to `vite.config.ts`
4. Replace CSS imports with Tailwind v4 entrypoint (`@import "tailwindcss"` in `src/index.css`)
5. Install shadcn/ui: `npx shadcn@latest init` ‚Äî select defaults, configure for Tailwind v4
6. Install radix primitives as needed by shadcn
7. Add a Button component via `npx shadcn@latest add button`
8. Render the Button in `App.tsx` to verify it works

## Verify
- `cd apps/podplay && npm run build` ‚Äî exits 0, no errors
