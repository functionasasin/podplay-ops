## Instructions
1. Create `src/components/dashboard/MetricsBar.tsx`.
2. Accept `projects` array as a prop.
3. Compute and display the following metrics:
   - **Total Projects**: count of all projects
   - **Active Deployments**: count of projects with status in active deployment statuses
   - **Revenue Pipeline**: sum of project revenue/value fields, formatted with `formatCurrency`
   - **By Status Breakdown**: count per `project_status` value, displayed as small pills/badges
4. Render as a horizontal bar of metric cards above the project list.
5. Use shadcn Card components for each metric.
6. Integrate `MetricsBar` into the dashboard route, passing the full (unfiltered) projects array.
