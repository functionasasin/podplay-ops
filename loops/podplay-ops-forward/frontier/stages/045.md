# Stage 045 — Intake Route Shell + Stepper UI

## Spec Files
- `ui-spec/wizard-intake.md`

## Instructions
1. Create `src/routes/projects/$projectId/intake.tsx` with a route component wrapped in `ProtectedRoute` and `AppLayout`.
2. Create `src/components/wizard/WizardStepper.tsx` — a reusable stepper component that accepts an array of step labels and a current step index.
3. The stepper must render 7 steps with labels pulled from the spec overview section: Customer Info, Venue Config, Service Tier, ISP Info, Installer, Financial Setup, Review & Submit.
4. Each step label is clickable and navigates to that step. The current step is visually highlighted (e.g., bold + accent color). Completed steps show a check indicator.
5. Wire the stepper into the intake route with local state tracking the current step index.
