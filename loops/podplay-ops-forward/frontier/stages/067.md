# Stage 067 — Cost Chain Calculation

## Spec Files
- `business-logic/cost-analysis.md`

## Instructions
1. Create `src/lib/cost-chain.ts`.
2. Export function `calculateCostChain(unitCost: number, quantity: number, taxRate: number, shippingRate: number, marginTarget: number): CostChainResult`.
3. Define the `CostChainResult` type with fields: `total`, `tax`, `shipping`, `landedCost`, `margin`, `customerPrice`.
4. Implement every formula from `business-logic/cost-analysis.md`:
   - `total = unitCost * quantity`
   - `tax = total * taxRate`
   - `shipping = total * shippingRate`
   - `landedCost = total + tax + shipping`
   - `margin = landedCost * marginTarget`
   - `customerPrice = landedCost + margin`
5. Use precise arithmetic — round to 2 decimal places only at the final output.
