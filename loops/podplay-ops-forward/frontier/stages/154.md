# Stage 154 — Responsive Tables

## Spec Files
- `ui-spec/responsive.md`

## Instructions
1. Wrap every `<table>` or DataTable component in a `div` with `overflow-x-auto` to enable horizontal scroll on small screens.
2. For key tables (projects list, BOM table, inventory table), create a card-view alternative that renders at the `sm` breakpoint and below.
3. Card view shows each row as a stacked card with label-value pairs.
4. Use `hidden sm:block` for table view and `sm:hidden` for card view.
