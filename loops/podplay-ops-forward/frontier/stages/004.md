# Stage 004 — Supabase Client + Local Docker Setup

## Spec Files
- `deployment/infrastructure.md`
- `data-model/schema.md`

## Instructions
1. `cd apps/podplay && npx supabase init`
2. Install Supabase client: `npm install @supabase/supabase-js`
3. Create `src/lib/supabase.ts` with `createClient` using env vars `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`
4. Create `.env.local` with local defaults:
   - `VITE_SUPABASE_URL=http://127.0.0.1:54321`
   - `VITE_SUPABASE_ANON_KEY=<local anon key from supabase start output>`
5. Add `.env.local` to `.gitignore` if not already present
6. Run `npx supabase start` to boot local Supabase (requires Docker)

## Verify
- `cd apps/podplay && npx supabase status` — shows running services (API, DB, Auth, etc.)
