# Stage 003 — TanStack Router Setup

## Spec Files
- `ui-spec/routes.md`

## Instructions
1. `cd apps/podplay`
2. Install TanStack Router: `npm install @tanstack/react-router @tanstack/router-plugin`
3. Configure the TanStack Router Vite plugin in `vite.config.ts` for file-based routing
4. Create `src/routes/__root.tsx` with a root layout containing `<Outlet />`
5. Create `src/routes/index.tsx` as the home route
6. Create a 404 catch-all route at `src/routes/$404.tsx` (or `__404.tsx` per TanStack conventions)
7. Update `src/main.tsx` to use `<RouterProvider>` with the generated route tree

## Verify
- `cd apps/podplay && npm run build` — exits 0, route tree is generated
