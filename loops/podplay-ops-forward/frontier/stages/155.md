# Stage 155 — Responsive Forms

## Spec Files
- `ui-spec/responsive.md`

## Instructions
1. Set all form inputs to `w-full` on mobile.
2. Convert multi-column form grids to `grid-cols-1` at the `sm` breakpoint and below.
3. Set minimum touch target height of `h-11` (44px) on all buttons, inputs, selects, and interactive elements.
4. Add `p-4` padding on the outermost form container for mobile screens.
5. Audit every form in the app (intake steps, project creation, settings forms, invoice forms) and apply these rules.
