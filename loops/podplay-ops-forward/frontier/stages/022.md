# Stage 022 — Auth Context + Provider + useAuth Hook

## Spec Files
- `ui-spec/routes.md`

## Instructions
1. Create `src/lib/auth.tsx` with an `AuthContext` using React `createContext`.
2. Implement `AuthProvider` component that wraps children and subscribes to Supabase `onAuthStateChange`. Track `user`, `session`, and `loading` in state.
3. Export a `useAuth()` hook that consumes `AuthContext` and returns `{ user, session, loading, signIn, signUp, signOut }`.
4. `signIn(email, password)` calls `supabase.auth.signInWithPassword`.
5. `signUp(email, password)` calls `supabase.auth.signUp`.
6. `signOut()` calls `supabase.auth.signOut` and clears local state.
7. Wrap the app's root in `AuthProvider` (update `main.tsx` or `App.tsx`).
