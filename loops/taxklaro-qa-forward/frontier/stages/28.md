# Stage 28 — Flow QA: Computation Flow

## Viewport
1280x800

## Instructions
Execute this flow using Playwright:

1. Sign in
2. Click "New Computation" on dashboard
3. Wizard step 1: select "Self-Employed Professional" → Next
4. Fill required fields through each step (use reasonable defaults)
5. Reach Review step → click Submit/Compute
6. Verify results render (recommendation banner, comparison table, breakdown)
7. Navigate to `/dashboard` → verify computation appears in recent list
8. Navigate to `/computations` → verify computation in list
9. Click computation → verify detail page loads with results

Screenshot key steps → `screenshots/flow-computation-{step}.png`

## Pass criteria
- Wizard completes without errors
- WASM computation succeeds
- Results save to Supabase
- Computation appears on dashboard and computations list
- Detail page shows saved results
