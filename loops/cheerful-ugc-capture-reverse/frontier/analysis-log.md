| 5 | 2026-02-28 | `ai-visual-detection` | 1 run | Three complementary sub-approaches: (1) YOLO logo detection — fine-tuned per-brand, 75-85% mAP, requires 200-500 labeled images/brand; (2) CLIP product recognition — zero-shot via embedding similarity, no brand-specific training needed, 65-75% top-1; (3) OCR — PaddleOCR v3.0 best for scene text, detects brand names as visible text. | 6 | 2026-02-28 | `ai-audio-detection` | 1 run | Four-stage pipeline: ffmpeg audio extraction (16kHz mono WAV) → STT transcription → brand mention extraction (NLP) → confidence scoring. STT options: Whisper (self-hosted, ~$0.0017/min), GPT-4o Mini Transcribe ($0.003/min cheapest API), AssemblyAI ($0.0042/min, LeMUR LLM extraction, 30% fewer hallucinations). Brand extraction: keyword/regex (zero cost, ~90% exact match) → fuzzy match (rapidfuzz, ~80% recall) → Claude Haiku NLP ($0.0003/transcript, handles homonyms). CRITICAL CHALLENGE: Background music degrades Whisper accuracy from ~20% WER to ~30-40% WER; Spleeter/Demucs audio source separation mitigates this. Audio detection is the CHEAPEST AI detection layer at scale ($0.003/video API vs $1.50/1K images for visual). Archive Radar confirmed to use audio STT + NLP for spoken brand mentions. No new infra for API path — Whisper API or GPT-4o Mini Transcribe is a simple API call. Self-hosted shares GPU with visual detection worker. Small effort overall (1 sprint for API path). |
