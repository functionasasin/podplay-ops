## Key Decisions Made

### Archive count: 24 (not 20)
The identity-synthesis estimated 20 archives (7 originals + 13 forks from signal-vs-noise pre-fork-audit). The fork-audit confirmed all 16 forks = ARCHIVE (including agent-skills, which signal-vs-noise tentatively kept). Plus clsandoval.github.io was confirmed ARCHIVE. Final count: 8 originals + 16 forks = 24.

### Profile README tone: lowercase, builder confidence
Chose the "lowercase + specific numbers" voice from identity-synthesis. Short sentences, no paragraphs, no badges/widgets. The "this readme was generated by a ralph loop" hook is earned (the loop actually exists) and sets the personality immediately.

### Company names in README and bio: YES
pymc-labs and nuts-and-bolts-ai are named explicitly in both the bio and README. The code is private; the companies are not. This is standard professional practice. Without these names, the profile has zero professional credibility signal (the code is all private).

### Pin slot #6: yolos-lph over agent-skills
agent-skills fork confirmed 0 original commits (fork-audit). yolos-lph completes the CV pipeline story, which is one of the two strongest public-facing narratives (agent infra being the other). yolos-lph wins slot #6.

### Execution script: gh CLI over raw curl
`gh` handles authentication transparently, handles pagination, and has better ergonomics for GraphQL. The script is idempotent (archiving an already-archived repo is a no-op, pushing unchanged README is a no-op).
