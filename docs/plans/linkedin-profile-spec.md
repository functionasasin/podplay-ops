## Appendix: Tone Guide

**Voice:** First person, present tense, builder confidence. Dinner-table register — how you'd explain your work to a sharp friend, not how you'd write a cover letter.

**YES:**
- "I built a custom MCP tool server at the protocol level — not a FastMCP wrapper."
- "The loop runs without me."
- "Master distribution rights across 4 countries."

**NO:**
- "I am passionate about leveraging AI to drive impact..."
- "Seasoned engineer with expertise spanning multiple domains..."
- "Proven track record of delivering innovative solutions..."

**Rules:**
1. Name things (Temporal.io, not "advanced workflow orchestration")
2. Specificity over adjectives (36K LOC, not "large-scale")
3. One sentence for conviction ("Bayesian reasoning matters more than fine-tuning")
4. Active verbs tied to outputs ("Built", "Architected", "Deployed", "Secured")
5. Never sell. Radiate "building things" not "available for work."
