## 5. Subscription Tiers and Features

**5.1 Free Tier (₱0/month).** The Free Tier provides:
- Unlimited anonymous Computations for users without an Account (subject to rate limits: 10 Computations per hour per IP address)
- Unlimited Computations for registered Free Tier users (subject to rate limits: 60 Computations per hour per Account)
- Comparison of all three tax regimes (8% Flat Rate, Graduated + OSD, Graduated + Itemized Deductions)
- Regime recommendation with savings display
- Quarterly estimate calculator
- Percentage tax estimate
- Penalty estimate
- Save up to 10 computations per month to Account history
- Access to public educational content
- No PDF export
- No API access
- No CPA client management

**5.2 Pro Tier (₱200/month or ₱1,999/year).** The Pro Tier provides everything in Free Tier plus:
- Unlimited computation saves (no monthly save limit)
- PDF export of computation results in BIR-compatible layout (up to 50 PDF exports per month)
- Quarterly tax tracker: schedule, track, and receive deadline reminder emails for 1701Q and 2551Q filings
- Annual computation history with year-over-year comparison
- BIR Form 2307 CWT tracker: store and manage up to 100 2307 entries per year
- Deadline calendar with email reminders (14 days and 3 days before each deadline)
- Priority email support (response within 2 business days)
- Rate limit: 300 Computations per hour per Account

**5.3 Enterprise Tier (₱1,499/month or ₱14,999/year).** The Enterprise Tier provides everything in Pro Tier plus:
- CPA client management: create and manage up to 50 client profiles
- Per-client computation history and PDF export
- Batch computation API: submit up to 100 taxpayer inputs in a single API call
- API access: full REST API with API key management (up to 5 active API keys)
- White-label PDF export: replace TaxKlaro branding with firm name and logo on all PDF exports
- White-label disclaimer header: add firm contact information to PDF exports
- Dedicated support channel (Slack or email, response within 1 business day)
- Rate limit: 1,000 Computations per hour per Account; API key rate limit: 500 requests per hour per key

**5.4 Feature Gating.** All features listed in Sections 5.2 and 5.3 as Pro or Enterprise-only are gated at both the API layer and the frontend. Attempting to access a gated feature through the UI will display an upgrade prompt. Attempting to access a gated API endpoint without the required subscription tier will return HTTP 403 with the body `{"error": "PLAN_INSUFFICIENT", "required_plan": "<tier>", "upgrade_url": "https://taxklaro.ph/upgrade"}`.
